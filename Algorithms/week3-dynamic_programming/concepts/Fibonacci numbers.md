- the *Fibonacci Sequence* is a numerical sequence such that:
- $F(0) = 0$
- $F(1) = 1$
- $F(n) = F(n-1) + F(n-2)$ if $n>1$
- this can be done recursively via divide and conquer, but is very inefficient
