- metadata is data about data
- the database server needs to store all of the following information
	- Table name
	- Column names
	- Column data types
	- Primary key columns
	- Index names
	- Indexed columns
	- Foreign key name
	- Foreign key columns

### information_schema
- the information_schema views available in MySQL are:
	- Schemata
	- Tables
	- Columns
	- table_constraints
	- schema_privileges
	- table_privileges
	- column_privileges
